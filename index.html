{G}roundwork

Groundwork uses best practice to create a foundation for building web applications. Promoting an OOCSS approach with it's powerful abstraction classes as well as module pattern scaffolding for Javascript.

  • Base stylesReset, Typography & Root Styling

    We use the reset of all resets by Eric Meyer. Some nice typograhical styles working off a consistent vertical rythym. As well as some simple button styles.

  • Abstraction classesClasses meant for extension

    With these powerful classes you can quickly build slick interfaces while writing a lot less code. Remember, these are open for extension; not modification.

  • Javascript scaffolding Colours & Sizing

    When you think reset you think Eric Meyer. His reset is just one of those things that does so much without you even knowing.

  • & The rest is up to you… Font Awesome, abstractions & other magic

    Everyone builds websites in their own way. Groundwork doesn't come with a grid system or any other packaged modules. Easily import more stylesheets when need be.
