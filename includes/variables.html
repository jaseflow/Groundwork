Variables are so much more than just easy ways to store colour values. Using both variables and mixins allow you to speed up development and make maintenance a breeze.

How to use variables in your CSS

Variables should be used to configure styles you have already put in place. They also allow you to create relative styles, meaning that certain visual styles will change depending on the value of a particular variable. This is extremely helpful when it comes to changing the theme of a website or create multiple skins for the same website. When done correctly you can change the complete look of a site just by changing one or two variables.

Rule of thumb - if there is any chance of a property changing in the future, turn it into a variable.
